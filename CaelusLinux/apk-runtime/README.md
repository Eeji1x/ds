# Caelus APK Runtime

This directory contains the APK-based runtime system for Caelus on Linux, inspired by Sober's approach for Roblox.

## Structure

- `apk/` - Caelus APK files
- `runtime/` - Android runtime configuration
- `config/` - Runtime configuration files
- `cache/` - Cached data

## How it works

This system creates a specialized runtime for Caelus's Android APK, allowing it to run on Linux without traditional Android emulation.

## Installation

Place your Caelus APK in the `apk/` directory and run the launcher.
